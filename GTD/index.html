GTD 搞定 读书笔记

1 五大步骤

1.1 收集（清空大脑）

Target: 一切实物赶出大脑（即：对一件事情不需要两次相同的思考）

  • 定期清空，且减少 Inbox 的数量
  • 外在：工作篮
  • 内在：大脑

1.1.1 处理

  • 丢弃
  • done
  • 委托
  • 提示信息
  • 明确目前重要的工作和任务

1.2 分析处理

Target: 决定下一步行动

When: 会后间隙，整理会议笔记

1.2.1 步骤

  • 这是什么？
  • 需要采取行动吗？
    • 否
      • 垃圾 删除
      • 目前没有打算 孵化
      • 潜在利用价值 加入参考资料
    • 是
      • 承诺完成哪些工作？需要达成哪些目标
      • 具体 下一步行动 是什么？

1.3 组织整理

Target: 生成清单

When: 长途旅行路上

1.3.1 形式

  • 项目清单 + 索引
  • 项目规划和资料库（视线之外触手可及）

1.3.2 结果

  • 每日行动
    • 日程表（约会，行程）
    • 下一步行动清单 ASAP
    • 等待清单 指派 @xxx
  • 资料
    • 需要酝酿 将来/也许 比如：书单
    • 参考资料 按主题分，一般性归档

1.4 检查回顾

Target: 确定下一步行动方案

When: 早上检查工作清单，回顾工作状况

1.4.1 方法 六层次

  • 人生
  • 3~5 年展望
  • 1~2
  • 责任
  • 当前项目/目标
  • 目前行动

1.4.2 成功的关键 - 每周回顾

  • 收集和加工所有的材料
  • 回顾整个系统
  • 更新各类清单

1.4.3 项目 List

  • Calender
  • NextActions
  • WaitingFor

1.5 执行

1.5.1 三种方法

  • 四标准
    • 情境
    • 有多少时间
    • 多少精力
    • 重要性
  • 三分类
    • 执行事先安排好的
    • 处理突发事件
    • 安排自己的工作

2 自然式计划法

  • 目标：这次会议的目的是什么？
  • 原则
  • 前景/结果

创造出清晰的结果： 超前预想工作的情况 想象 大获全胜 时的景象 捕捉预想中有益于成功的各种特点/方向/品质

  • 头脑风暴/集思广益
  • 组织整理 主次/先后/轻重
  • 下一步行动 可以问人下面怎么处理

3 非自然法

  • 不仅忙着做事，而要站在旁边静静想一想

3.1 如何新建任务

  • 这项工作的预期成果是什么？
  • 下一步的具体行动到底是什么？
  • 将上面二者填入 GTD 系统

4 实践

4.1 摘录

  • 电话转到语音信箱
  • 9 分钟简历一个紧急工作区
    • 门作桌台架于文件柜
    • 三个工具盒
    • 记事本+笔
  • 移动办公区
  • 归档系统 - 至少一年整理一次
  • 宏观：项目的定义，承诺、目标 – 可以作为 heading
  • 微观：下一步行动

4.1.1 行动

  • 项目清单 + 规划和资料的存储系统
  • 日程表
  • 下一步行动清单
  • 等待清单

4.2 清单

从下一步行动列表归纳出的

  • 电话
  • 在电脑旁
  • 外出事宜
  • 在办公室
  • 在家时
  • 议事日程 - 需要和别人交流
    • 按会议分
    • 按人员分
  • 阅读/回顾
  • 等待清单
    • 需与 NextActions 放入同一系统，才能取得最佳效果

4.2.1 项目清单分类

  • 个人/职业
  • 委托给他人的项目 - 等待清单
  • 特定类型项目

4.2.2 将来/也许清单

  • 学习技能
  • 创意
  • 旅行
  • 购物
  • Links 要浏览的网站

4.3 日程表

  • 提醒自己落实某个项目
  • 可能要参与的活动
  • 提醒自己作出决定
  • （需要指定时间获取的信息）

4.4 备忘录系统 43Folds

每天将下一天的文件夹中的内容放入工作篮，再把这个文件夹排列到后面，每日更新

4.5 核查清单 - 完成后清空

  • 个人主张
  • 工作责任范畴
  • 旅行核查清单
  • 每周回顾
  • 会议前检查
  • 生活/工作中关键任务
  • 组织结构
  • 个人发展

4.6 每周回顾

  • 项目 （包括项目规划）
  • 下一步行动
  • 议事日程
  • 等待列表
  • 将来/也许
  • 脑无牵挂
